- Descrição do Tema do Projeto: O projeto visa desenvolver uma Aplicação Web Responsiva que consome dados sobre a pandemia de COVID-19 de APIs públicas. A aplicação terá um Design interativo, apresentando informações atualizadas sobre casos, recuperações e vacinas, por meio de gráficos, tabelas e mapas, facilitando a visualização e análise dos dados para usuários e pesquisadores.

-Especificação das Tecnologias e Ferramentas

Front-end: - HTML/CSS: Para a estruturação e estilo da página. - JavaScript: Para interatividade e construir uma interface de usuário dinâmica. - Chart.js: Para visualização de dados em gráficos. *Back-end: -.js: Para o desenvolvimento do servidor. *APIs: - API COVID-19: Para consumo dos dados atualizados sobre a pandemia.

- Créditos sobre a Origem dos Dados:

Os dados utilizados na aplicação serão obtidos a partir de APIs públicas, que disponibilizam informações atualizadas e verificadas sobre casos de COVID-19, mortes e vacinação em diversos países e regiões.
